# Design System - Mister Churras

## 1. Visão Geral
O app deve exalar **autoridade** e **rusticidade premium**. O visual deve ser escuro (dark mode nativo), com contrastes fortes e interface direta, tratando o usuário como o verdadeiro guardião da brasa.

## 2. Paleta de Cores
O sistema de cores utiliza três pilares fundamentais:
- **Carvão (Primária / Fundo):** `#1A1A1A`
  - *Uso:* Fundo principal, painéis grandes, garantindo o visual rústico noturno.
- **Brasa (Ação / Destaque):** `#D35400`
  - *Uso:* Botões principais de chamada para ação (Call to Action), ícones de destaque, alertas e barras de progresso.
- **Off-white (Secundária / Texto):** `#F4F4F4`
  - *Uso:* Tipografia primária, ícones secundários, bordas sutis e cards em sobreposição ao Carvão.

## 3. Tipografia
Recomenda-se o uso de fontes sem serifa modernas e robustas (ex: *Inter*, *Roboto* ou *Oswald* para títulos épicos).

## 4. Elementos UI (Mobile-First)
- **Botões:** Devem ser grandes e fáceis de tocar (mínimo de 48px de altura). Devem apresentar transições rápidas de estado (hover, active) para feedback imediato.
- **Inputs e Controles:** Controles de incremento/decremento `(+) / (-)` devem ser proeminentes para rápida inserção de número de convidados ou quantidade de horas.
- **Feedback Tátil & Visual:** Uso de micro-interações. Sempre que o usuário adicionar carne ou convidados, uma breve animação ou sombreado (glow) estilo "brasa" deve ocorrer no botão.

## 5. Copywriting e Tom de Voz
- **Tom:** Épico, direto, de Mestre para Mestre.
- **Exemplos de Copy:**
  - "Inicie a Brasa" (em vez de "Começar")
  - "Guardiões" (em vez de "Homens")
  - "Lista do Mestre" (em vez de "Lista de Compras")
