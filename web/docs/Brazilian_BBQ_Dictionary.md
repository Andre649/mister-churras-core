# Dicionário do Braseiro (Brazilian BBQ Dictionary - Padrão Taurus)

Este é o dicionário oficial da aplicação. Todo o código, UI/UX e comunicação deve evitar estrangeirismos e "gourmetização". A linguagem é direta, focada na experiência do balcão nacional.

## 🔪 Mapeamento Oficial de Termos

| Termo Legado (Estrangeiro/Gourmet) | Termo Oficial do Braseiro (Nacional) | Contexto / Uso |
| :--- | :--- | :--- |
| **Mister Churras Chronicles** | **Mister Churras: O Braseiro** | Nome oficial do aplicativo / Marca |
| **Event / Events** | **Estação de Corte / Estações** | Um churrasco criado pelo usuário no app. Ex: "Criar nova Estação de Corte" |
| **Split / Bill / Check** | **Rateio** | Divisão de custos entre os convidados. Ex: "Resumo do Rateio" |
| **Calculation / Engine** | **Matemática do Fogo** | O algoritmo que calcula as quantidades de insumos. |
| **Supplies / Groceries** | **Insumos / Bebidas e Carnes** | Os itens calculados para o evento. |
| **Dashboard / Home** | **Quartel-General / Acampamento** | A tela principal do usuário. (Usar "Acampamento" para a home). |
| **Login / Sign In** | **Entrar na Roda / Acessar** | Tela de entrada. |
| **Sign Up / Register** | **Assinar o Ponto / Chegar no Braseiro** | Tela de cadastro. |
| **Acender o fogo** | **Acender o braseiro** | Ação de iniciar a organização/sistema. |
| **Selar o destino** | **Montar lista** | Ação de confirmar e fechar a lista de rateio/compras. |
| **Tipo (de evento)** | **Tamanho da churrasqueada** | Referência ao porte ou estilo da celebração. |

## ⚠️ Regras de Ouro
1. **Sem Frescura:** A linguagem deve ser seca e direta.
2. **Respeito ao Fogo:** A comunicação deve invocar o respeito aos grandes festivais de carne do Brasil (como o Festival Taurus).
3. **Rusticidade:** O visual acompanha o texto. Nada de neon ou bordas arredondadas. O peso da prensa de 1890.
